개요

이 매뉴얼의 목적

이 매뉴얼은 비개발자가 AI Agent와 함께 업무에 필요한 문서나 작은 프로그램을 만들고, Confluence, HTML, 클라우드 서버 등 알맞은 방식으로 공유하거나 사용할 수 있도록 돕는다.

아래 흐름도는 업무 아이디어가 문서나 프로그램으로 만들어지고, 어디에 저장하고 공유되는지 간단히 보여준다.
