Atlassian MCP 서버 설치 (수정예정)

목적

Atlassian MCP 서버는 Claude Code가 Confluence나 Jira 같은 Atlassian 도구에 접근할 수 있게 연결하는 통로다. 문서 초안 작성, 요구사항 정리, 이슈 확인처럼 협업 도구의 정보를 참고해야 할 때 사용한다.

설치 전 확인할 것

  • Confluence 또는 Jira 접근 권한이 있는지 확인한다.
  • 조직에서 허용한 인증 방식과 토큰 발급 방법을 확인한다.
  • Claude Code에 연결해도 되는 문서 범위를 사전에 정한다.

진행 순서

  1. Atlassian MCP 서버 설치 방법을 확인한다.
  2. 필요한 인증 정보와 환경 변수를 준비한다.
  3. Claude Code 설정에 Atlassian MCP 서버를 추가한다.
  4. Confluence 또는 Jira 문서 검색이 가능한지 테스트한다.

테스트 프롬프트 예시

Atlassian MCP 서버 연결 상태를 확인해줘.
Confluence에서 접근 가능한 문서 범위를 요약하고, 매뉴얼 작성에 사용할 수 있는 자료가 있는지 알려줘.
문서 원문 전체를 출력하지 말고 제목과 활용 가능성 중심으로 정리해줘.
