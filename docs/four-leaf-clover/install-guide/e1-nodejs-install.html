node.js 설치하기 (수정예정)

Node.js가 필요한 이유

Node.js는 터미널에서 특정 프로그램을 실행할 수 있게 도와주는 기본 실행 환경이다. 쉽게 말하면 Claude Code CLI나 MCP 서버 같은 도구를 PC에서 설치하고 실행하기 위해 필요한 준비물 중 하나라고 이해하면 된다.

npm은 Node.js와 함께 설치되는 도구다. 필요한 프로그램을 내려받고 설치하는 역할을 한다.

Node.js 설치 여부 확인하기

  1. 터미널을 연다.
  2. 아래 명령어를 입력하고 Enter를 누른다.
node -v

v20.11.1처럼 v로 시작하는 숫자가 보이면 Node.js가 설치되어 있는 상태다. 숫자는 PC마다 다를 수 있다.

command not found: node 또는 비슷한 메시지가 보이면 Node.js가 설치되어 있지 않거나, 터미널에서 Node.js를 찾지 못하는 상태다.

npm 설치 여부도 함께 확인하기

  1. 같은 터미널 화면에서 아래 명령어를 입력한다.
npm -v

10.2.4처럼 숫자가 보이면 npm도 사용할 수 있는 상태다. npm은 이후 Claude Code CLI 또는 MCP 서버를 설치할 때 사용할 수 있다.

npm과 npx의 차이

npm은 필요한 프로그램을 PC에 설치할 때 사용하는 도구다. 예를 들어 어떤 도구를 계속 사용할 예정이라면 npm으로 설치해두고 필요할 때마다 실행할 수 있다.

npx는 프로그램을 매번 직접 설치해두지 않고, 필요할 때 바로 실행하는 도구에 가깝다. 설치 과정 없이 한 번 실행해보거나, MCP 서버처럼 특정 연결 도구를 바로 실행할 때 자주 사용된다.

간단히 말하면 npm은 “설치하기”, npx는 “바로 실행하기”에 가깝다.
