Expanded 3

GQuery MCP 서버 설치 (수정예정)

목적

GQuery MCP 서버는 Claude Code가 GQuery에 있는 정보를 작업에 활용할 수 있게 연결하는 통로다. 비개발자 입장에서는 Claude Code가 필요한 내부 데이터를 직접 참고할 수 있게 해주는 준비 단계라고 이해하면 된다.

설치 전 확인할 것

  • GQuery 접근 권한이 있는지 확인한다.
  • Node.js와 npm 또는 npx를 사용할 수 있는지 확인한다.
  • 인증 토큰이나 계정 정보가 필요한 경우 안전한 저장 위치를 확인한다.

진행 순서

  1. GQuery MCP 서버 설치 방법과 필요한 환경 변수를 확인한다.
  2. Claude Code 설정에 GQuery MCP 서버를 추가한다.
  3. 연결 테스트를 실행해 Claude Code가 GQuery 정보를 읽을 수 있는지 확인한다.
  4. 테스트 프롬프트로 필요한 정보를 가져오는지 검토한다.

테스트 프롬프트 예시

GQuery MCP 서버 연결 상태를 확인해줘.
접근 가능한 데이터 범위를 요약하고, 매뉴얼 작성에 참고할 수 있는 정보가 있는지 알려줘.
민감정보는 출력하지 말고, 어떤 종류의 정보에 접근 가능한지만 설명해줘.
